# SaaStral - Domain Language

When discussing features or implementing code, use this vocabulary:

| Term | Meaning |
|------|---------|
| **Offboarding** | When an employee leaves the company |
| **Orphaned License** | License still assigned to someone who left |
| **Seat** | A single user license (e.g., "Slack has 20 seats") |
| **Usage Rate** | Percentage of paid seats actually being used |
| **Adoption** | How many people are using a tool vs. total seats |
| **Renewal** | When a subscription payment recurs |
| **Shadow IT** | Software purchased without IT/Finance approval |
| **SaaS Stack** | All the SaaS tools a company uses |
| **Duplicate Tool** | Multiple tools that do the same thing |
